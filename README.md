# BookYourBooth - Exhibition Booth Booking System

A modern exhibition booth booking system built with Next.js, TypeScript, Tailwind CSS, and a Node.js backend. Supports user authentication, role-based access, and real-time booth availability.

## Features

### Members
- Register & login
- Browse & filter exhibitions
- Multi-step booth booking
- Manage personal bookings

### Admins
- Manage exhibitions
- View & filter bookings
- User management & role control
- Dashboard with statistics

### Design
- Modern, minimal dashboard style
- Mobile-first & responsive
- Smooth animations & accessible colors

## Tech Stack
- **Frontend:** Next.js 16, TypeScript, Tailwind CSS, shadcn/ui, Lucide React  
- **Backend:** Node.js, Express.js, MongoDB, JWT, bcryptjs  
- **Security:** helmet, xss-clean, rate limiting  
- **Docs:** Swagger UI
